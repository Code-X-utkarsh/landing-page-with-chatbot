# SDE Outside India - Landing Page

A high-performance, immersive 3D landing page built with Next.js 14, Three.js, and Tailwind CSS.

## Features

- **3D Interactive Elements**: Pixelated Globe, Floating Character, Interactive Laptop.
- **Scroll Animations**: Integration of GSAP ScrollTrigger for smooth storytelling.
- **Responsive Design**: Mobile-optimized 3D scenes and layout.
- **Design System**: Custom Tailwind configuration based on the provided design tokens.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS v4
- **3D Engine**: React Three Fiber (Three.js)
- **Animation**: GSAP, Framer Motion
- **Smooth Scroll**: Lenis

## Getting Started

1. **Install Dependencies**

```bash
npm install
```

2. **Run Development Server**

```bash
npm run dev
```

3. **Build for Production**

```bash
npm run build
```

## Project Structure

- `src/components/three`: 3D scenes and objects
- `src/components/sections`: Page sections (Hero, About, etc.)
- `src/app/globals.css`: Design system variables and global styles
